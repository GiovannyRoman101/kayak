
# Kayak

It is the kayak online assignment that renders all flights and filters them by alliances




## Run Locally

Clone the project

```bash
  git clone https://github.com/GiovannyRoman101/kayak.git
```

Go to the project directory

```bash
  cd kayak
```

Install dependencies

```bash
  npm install
```

Start the web page

```bash
  npm run start
```
